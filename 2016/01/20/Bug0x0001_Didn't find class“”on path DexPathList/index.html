#0x0001_Didn’t find class“”on path DexPathList
Logcat提示：

Didn't find class " " on path: DexPathList

BUG症状
在Android 5.0以下的系统上无法运行程序，而在Android5.0以上的系统中则正常运行
可能原因
经过几次Google之后，在Stackoverflow上找到了可行的方案，造成这个Bug的可能原因是依赖库工程的android-support-lib的版本（包括design包下的库）不一致导致的，将各依赖库的android-support-lib修改一致后Bug解决
